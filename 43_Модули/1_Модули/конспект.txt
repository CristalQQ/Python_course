Модули позволяют структурировать код и разбить его на разные части
Модули позволяют избегать дублирования блоков кода
Любой файл с расширением .py является модулем в python

Для импорта из дуругого модуля нужно исползовать import
Чтобы переименовать модуль используется as
Для импорта определенных пременных из модуля - from module_one import print_name, my_fn
